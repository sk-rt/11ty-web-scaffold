# 11ty website development scaffold

- eleventy v1.0
- webpack v5.6
- typeScript v4.5
- scss v1.4

## Setup & Configuration

edit .env.\*

- .env.local: for local development
- .env.production: for production build

install node modules

```sh
yarn
```

## Development server

```sh
yarn dev
```

## Production build

```sh
yarn build
```
